# 🌿 Paradise Nursery Shopping Cart App

A modern and responsive shopping cart web application built using **React, Redux, and Vite**.
This app allows users to browse products, add them to a cart, and manage their shopping experience efficiently.

---

## 🚀 Features

* 🛒 Add and remove items from the cart
* 🔄 Real-time cart updates
* 📦 Quantity management
* ⚡ Fast UI powered by Vite
* 📱 Responsive design (mobile-friendly)

---

## 🛠️ Tech Stack

* **Frontend:** React
* **State Management:** Redux
* **Build Tool:** Vite
* **Styling:** CSS

---

## 📦 Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/ayushsanwal/paradise-nursery-app.git

# Navigate to project folder
cd paradise-nursery-app

# Install dependencies
npm install

# Run development server
npm run dev
```

👉 Open in browser:
http://localhost:5173
